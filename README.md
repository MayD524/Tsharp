WARNING! THIS LANGUAGE IS A WORK IN PROGRESS! ANYTHING CAN CHANGE AT ANY MOMENT WITHOUT ANY NOTICE!

> Something like Forth and Porth, but in C.

### Install

> Install
```
make
```

### Run

> Run
```
$ ./tsh.out <filename>.t#
```

> Hello World
```pascal
push::"Hello World":

print:
```

> dup example
```pascal
push::"Hello World": dup: print: print:
```
